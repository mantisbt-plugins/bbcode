MantisBT BBCode Plugin
My first Plugin for MantisBT.
It's help convert some bbcode to html-style.

Version see in VERSION.txt

Installation see in INSTALL.txt

Supported bbcode:
	[b]      => <strong>
	[i]      => <i>
	[u]      => <u>
	[del]    => <s>
	[sub]    => <sub>
	[sup]    => <sup>
	[tt]     => <tt>
	[img]    => <img>
	[url]    => <a href>
	[left]   => <div align>
	[right]  => <div align>
	[center] => <center>
	[hr]     => <hr>
	[color]  => <span style>
	[code]   => <pre><code>
